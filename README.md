# eduTrack
School Management System

Overview

School Management System is a web-based application developed using Flask that helps manage student academic records efficiently. The system provides an admin dashboard to store student information, calculate grades automatically, track attendance, view performance statistics, and analyze student data through interactive charts.

Features

- Secure Admin Login
- Add and Manage Student Records
- Automatic Percentage and Grade Calculation
- Attendance Management
- Student Search Functionality
- Grade-Based Filtering
- Dashboard Statistics
- Performance Analytics with Charts
- Responsive User Interface
- Dark Mode Support

Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- Chart.js
